Numpy 패키지의 함수와 속성
-배열 속성
	dtype : 요소들의 타입 (타입변경시 astype(타입)이용)
	itemsize : 요소의 byte사이즈
	ndim : 차원수(축수)
	size : 요소 개수
	shape : 배열의 구조 (tup)
-배열 생성 함수
	np.array(obj,dtype,copy=T/F
	np.array(obj,dtype,copy=T/F reshape(행수, 열수)
	np.zeros(tup,dtype)
	np.ones(tup,dtype)
	np.full(tup, 초기값, dtype)
	np.empty(tup,dtype)

	※n차원 배열 만들기
	- np.random.random( tup ) : 0부터 1 미만의 균등 분포를 갖는 tup shape 사이즈 n차원배열
	- np.random.uinform(low=0, high=1.0, size=tup) : low~ high 미만까지 균등분포 n차원배열 
	- np.random.normal(loc=0.0, scale=1.0, size=tup) : 평균 loc, 표준편차scale을 갖는 정규분포n차원배열
	
	※1차원 배열 만들기
	- np.random.rand(n) : 균등분포의 1차원 0~1미만 난수 1차원 배열 
	- np.random.randn(n) :  표준정규분포(평균0, 표준편차가 1인 정규분포)의 1차원 배열 
	
	- np.arange(from, to, by, dtype): from~to 앞까지 by씩 증감하는 1차원 배열
	- np.linspace(from,to,num) : from~to까지 같은 간격의 num개 수를 1차원 배열로  

-배열 차원 변경
	reshape(): -1 사용하면 자동계산
	resize(): -1불가
	T, transpose(): 전치행렬
	flatten(), ravel(): 1차원으로 변경

-배열 연산
	요소별연산, 행렬곱(@) * 단위행렬 A@B ==B@A
        집계함수(sum, min, max, mean, var, std)
	범용함수(np.add, np.sqrt,....), 브로드케스팅

- 배열 합치기
	np.vstack( tup )
	np.hstack( tup )
	np.dstack( tup ) : 마지막 축(열)들끼리 합침
	np.concatenate( tup, axis) axis축으로 합침

	np. column_stack( tup ) : 1차원배열을 열단위로 배열하여 2차원 배열로 합침 
						dstack에 차원하나 뺀것과 같다
	np.row_stack( tup ) : 행단위로 합침

	np.stack(tup, axis)

	np.r_[a, b , c] row단위로 쌓음(한행으로 합치기)
	np.c_[a, b, c] column으로 쌓아서 합치기 	(1차원을 2차원으로
	1차원배열[:, np.newaxis] : 1차원 배열을 2차원으로 쌓아줌(축 증가)
		말고도 reshape(-1,1) / np.expeand_dims(a, axis=1) 쓸수 있다

- 배열 분할하기
	vsplit(array,indices_or_sections) : 첫번째 축(2차원의 경우 가로축)을 따라 분할
	hsplit(array,indices_or_sections) : 두번째 축(2차원의 경우 세로축)을 따라 분할
	split(array,indices_or_sections,axis) : axis가 0이면 vsplit, axis가 1이면 hsplit과 동일

	array_split(array,indices_or_sections, axis) : 나눠지지 않는 정수로도 split 가능
