Numpy 패키지의 함수와 속성
-배열 속성
	dtype : 요소들의 타입 (타입변경시 astype(타입)이용)
	itemsize : 요소의 byte사이즈
	ndim : 차원수(축수)
	size : 요소 개수
	shape : 배열의 구조 (tup)
-배열 생성 함수
	np.array(obj,dtype,copy=T/F
	np.array(obj,dtype,copy=T/F reshape(행수, 열수)
	np.zeros(tup,dtype)
	np.ones(tup,dtype)
	np.full(tup, 초기값, dtype)
	np.empty(tup,dtype)
